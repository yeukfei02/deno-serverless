## Requirement:

- install yarn
- install node (v12+)
- install serverless
- install deno

## Testing and run:

```
// deploy to serverless
$ yarn run deploy

// open serverless dashboard
$ yarn run dashboard

// run test case
$ yarn run test

// remove serverless services in aws (api gateway, lambda, s3, cloudformation)
$ yarn run remove
```
